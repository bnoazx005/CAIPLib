Типы

typedef struct TColor TColor

Структура, содержащая информацию о пикселе изображения.

В библиотеке все изображения имеют формат RGBA с 8-ю битами на компоненту. Альфа компонента (A) не используется.

Для grayscale и binary изображений значения компонент цветов равны между собой. Порядок членов структуры обусловлен особенностями хранения данных в TGA файлах.

Тип caUint8 - переопределение unsigned char.

typedef struct TImage TImage

Структура, содержащая информацию об изображении

Поддежриваются только несжатые TGA изображения с 32-битным форматом пикселей.

typedef struct TTGAHeader TTGAHeader

Структура, описывающая заголовок TGA файла.

Поддежриваются только несжатые TGA изображения с 32-битным форматом пикселей.

Перечисления

enum E_EDGE_DETECTOR_TYPE

Перечисление типов детекторов границ, реализованных в библиотеке

Элементы перечислений
EDT_LUM_DIFF 

Тип детектора, учитывающий разницу яркости текущего пикселя с соседними

EDT_ADV_LUM_DIFF 

Тип детектора, учитывающий разницу яркости между парами соседей, значение текущего пикселя не учитывается

EDT_GRADIENT 

Детектор, вычисляющий величину вектора градиента для каждой точки

enum E_ERROR_TYPE

Перечисление кодов ошибок, которые могут возникать в процессе работы функций

Элементы перечислений
ET_SUCCESS 

Успешное завершение работы

ET_INVALID_ARGUMENT 

Некорректное значение одного или более аргументов

ET_BAD_ALLOCATION 

Ошибка выделения памяти

ET_INVALID_FILE_OPERATION 

Ошибка, возникшая при выполнения файловой операции (чтение, запись, создание, открытие и т.д.)

ET_INVALID_TGA_TYPE 

Тип считываемого TGA файла не соответствует требованиям библиотеки

ET_FAIL 

Критическая ошибка, продолжение работы невозможно

ET_INVALID_IMAGE_SIZES 

Некорректные размеры изображения (не удалось сконфигурировать сетку блоков для CUDA вычислителя)

ET_MEMORY_LEAK_DETECTED 

Зафиксирована утечка памяти. Возможно, где-то произошла ошибка освобождения памяти

ET_UNKNOWN_EDGE_DETECTOR_TYPE 

Неизвестный тип детектора границ

ET_UNKNOWN_NOISE_FILTER_TYPE 

Неизвестный тип фильтра шума

ET_UNKNOWN_ERROR 

Иная ошибка

enum E_NOISE_FILTER_TYPE

Перечисление типов фильтров шума

Элементы перечислений
NFT_PEPPER 

Тип фильтра, удаляющий серый и черный шум

NFT_SALT 

Тип фильтра, удаляющий белый шум

NFT_SALT_AND_PEPPER 

Тип фильтра, удаляющий сочетание серых и белых шумов

NFT_MEDIAN 

Медианный фильтр
