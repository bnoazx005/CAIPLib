dim3 &  gridDim,
dim3 &  threadsDim 
)

Функция, конфигурирующая сетку блоков для CUDA вычислений

Аргументы
[in]currImageИзображение
[out]gridDimРазрешение сетки блоков
[out]threadsDimРазрешение одного блока (количество потоков по осям X и Y для одного блока)
Возвращает
Возвращает код ошибки, который можно передать в функцию caCheckError для проверки на корректное выполнение. В случае если не удалось подобрать конфигурацию, возвращает код ошибки ET_INVALID_IMAGE_SIZES
