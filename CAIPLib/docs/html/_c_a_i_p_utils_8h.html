Подробное описание

Заголовочный файл CAIPUtils.h содержит объявления вспомогательных стредств.

Авторы
Касимов Ильдар
Дата
14.07.2015

Данный файл является дополнительным и содержит объявления различных вспомогательных средств, которые рекомендуется использовать при работе с данной библиотекой.

Функция, возвращает цвет пикселя в указанных координатах.

Функция предназначена для выполнения на графическом процессоре. В отличие от GetColor, также возвращает индекс в одномерном массиве пикселя с указанными координатами.

[in]imageHeightВысота изображения

Возвращает
Возвращает цвет пикселя в указанных координатах. В случае, если x или y выходят за допустимые пределы, функция вернет черный цвет.

Функция, возвращающая наибольшее из двух значений

Функция не использует условных операторов, основана на арифметических вычислениях

\[ max(a, b) = \frac{(a + b + \left|a - b \right|)}{2} \]

Аргументы
[in]aПервый аргумент
[in]bВторой аргумент
Возвращает
Возвращает наибольшее из двух аргументов
